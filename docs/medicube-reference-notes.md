# Medicube Reference Notes

Reference URL: https://themedicube.co.kr/

## Observed Patterns

The site uses a black promotional top bar, a white navigation header, strong black Korean typography, large visual commerce cards, circular/compact category shortcuts, red sale or status badges, and product cards where price and membership value are prominent.

The navigation and content taxonomy are especially relevant because the site exposes device products, device comparison, device usage, skincare usage, community, customer support, and authentic product registration inside a Cafe24 commerce structure.

## CuraPulse Translation

CuraPulse should borrow the commerce clarity and large visual rhythm, but not the exact look. The color system should shift from strong sale red toward Pulse Coral, clinical white, and light mint. The page hierarchy should make SCIENCE and routine logic more visible than ordinary event commerce.

## Design Decisions

The prototype should stay desktop only for v0.1. It should not include pop-up heavy behavior, real payment, real search, or real member logic. Device Auth, ROUTINE LAB, PULSE PICK, and SCIENCE should be visually represented as modules even if they remain mock-only.
