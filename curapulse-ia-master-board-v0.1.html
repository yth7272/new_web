01

Strategy Bridge

마케팅 대전제(ON/IN/MIND)와 자사몰 실제 GNB는 다른 언어로 다른 청중에게 말한다. 둘을 명시적으로 연결.
피부 위에서 작동하는 케어디바이스 + 외용 화장품을 통한 즉각적 케어
ON
몸 안에서 작동하는 케어먹는 뷰티 · 이너뷰티 · 흡수 케어
IN

MIND
⟶

ON / IN / MIND는 고객에게 세계관을 이해시키는 마케팅 언어이고, DEVICE / BOOSTER / INNER / CALM은 자사몰에서 실제 클릭하는 시스템 메뉴다. 메인 페이지 상단에서 ON/IN/MIND를 한 번 설명한 뒤, 클릭 가능한 메뉴는 시스템 명칭으로 노출한다.
회의 결정 (조남규 대표): "마케팅하고 실제 카테고리는 저기 정해진 대로 가면 돼요. 이건 대전제라는 거죠." — 함범주 본부장 지적: "일반 소비자가 봤을 때 컨셉 이해 난이도가 있다." → 메인 페이지에 ON/IN/MIND 설명 블록 필수.

Multi-Entry Matrix

라인명을 모르는 신규 고객도 친숙한 키워드로 진입 가능. 카테고리는 4가지 진입 방식을 모두 제공.
라인별
슬로건 패키지 단위
BOO데일리 글로우 BOO펌업 BOO라이트 BOO베어 BOO클리어 BOO컴포트 INN콜라겐 · 글로우 · 디톡스 · 안티에이징 · 슬립 CALM슬립 · 무드 · 데이 · 텐션 · 아로마
고민별
카테고리 횡단 큐레이션
탄력 · 리프팅 → 헤일로 + 펌업 + 콜라겐 미백 · 톤업 → 디바이스 + 라이트 + 글로우 진정 · 민감 → 베어 + 캄 슬립 슬립 · 스트레스 → 캄 전 라인 + 이너 슬립 모공 · 트러블 → 클리어 + (향후) 부스터 클렌저
유형별
제형 · 형태 필터
외용토너 외용앰플 외용세럼 외용크림 외용패드 외용마스크 외용클렌저 외용미스트 이너캡슐 · 분말 · 액상 · 젤리 디바이스페이셜 · 바디 · 액세서리
선물하기
시즌 매출 · 객단가
시즌홀리데이 · 명절 · 어버이날 · 화이트데이 · 크리스마스 가격대~5만 · ~10만 · ~20만 · 20만~ · 50만~ 대상엄마 · 여친 · 친구 · 나에게 · 신혼 옵션포장 · 카드 · 익명 발송 · 영수증 분리
외주 카페24 개발자에게 핵심: 다중 진입로는 단순 카테고리 등록이 아니라 상품 태그 · 필터 · 진열 룰 · 중복 노출 정책이 필요. 한 SKU가 여러 진입로에 동시 노출되는 구조 검증 필요.
